FAQs

Getting Started

Welcome! To get started making a new free body diagram select "New Diagram" from the menu. Although this application was designed to be used with the Myo Gesture Control Armband, it can also be used with a mouse.

Draw an Arrow

Simply select the arrow button and the size you want it to be. Select the new arrow and position it where you want. Don't forget to label the arrow as well using the "Apply Label" button.

Drawing a Shape

Similar to drawing an arrow, shapes or predrawn images can be placed and then resized or repositioned on the canvas. Just select the shape to be placed and the size you want to use.

Load an Image

Custom images can be loaded into the application using the "Load Image" button in the diagram editor. The application will remember all previously uploaded images.

Shapes and Images with Keyboard

Shape and image objects can be manipulated with the keyboard too.

To toggle selection of objects, press 'n'

To clear seletion (make nothing selected), press 'c'
