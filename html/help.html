FAQs

Getting Started

Welcome! To get started making a new free body diagram click or select "New Diagram" from the Menu. Also you can choose to begin from a previous diagram using "Load Diagram". Although this application was desgined to be used with the Myo Gesture Control Armband, it can also be used with a mouse.

Draw an Arrow

In order to draw an arrow using the editor, simply select the draw arrow button and position the arrow where you want and make sure it points in the correct direction. Don't forget to label the arrow as well using the "Apply Label" button.

Drawing a Shape

Similar to drawing an arrow, predrawn images can be placed and then resized or positioned on the canvas. Just select the shape to be placed, and begin editing.

Load an Image

Custom images can be loaded into the application using the "Load Image" button in the diagram editor. Either start a new diagram or load a previous diagram then select the button. The application will remember all previusly uploaded images.

Shapes and Images with Keyboard

Shape and image objects that have been added to the diagram can be manipulated with the keyboard.

  • To toggle selection of objects, press 'n'
  • To delete a selected object, press the backspace key
  • To clear seletion (make nothing selected), press 'c'
